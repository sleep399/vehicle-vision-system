# 登录账号说明

系统提供账号密码、邮箱/手机号验证码和微信扫码三种登录方式。

## 已提供的演示账号

| 用户名 | 密码 | 说明 |
| --- | --- | --- |
| `admin` | `admin123` | 系统首次启动时自动创建的管理员演示账号。 |

## 其他账号来源

- **注册账号**：在登录页的“注册”标签输入用户名、密码，以及可选邮箱或手机号；注册完成后会自动登录。
- **验证码账号**：输入邮箱或手机号、获取验证码并登录。首次成功登录时会创建账号，用户名取邮箱 `@` 前的部分或手机号本身。
- **微信扫码演示账号**：扫码确认后按需创建 `wechat_demo_user`。这是课程演示流程，不对应真实微信身份，也没有可供密码登录的密码。

密码只以 bcrypt 哈希形式保存，系统无法也不会输出已注册用户的密码。

> 验证码当前为演示模式：页面弹窗会显示验证码。实际部署时应配置 SMTP/SMS 服务，并禁止在接口响应中返回验证码。
